# ChatGPT에게 일을 제대로 시키는 법 배우기

## 프로젝트 개요

본 프로젝트는 LLM을 활용하여 반복적인 업무 문서 작성 과정을 자동화하기 위한 프롬프트 패키지를 설계하고 검증한 과제이다.

선택한 업무 과업은 **데이터 분석 결과 기반 보고서 개요 생성**이다. 사용자가 분석 목적, 데이터 범위, 주요 수치, 관찰된 패턴을 입력하면, LLM은 이를 바탕으로 보고서 제목, 한 줄 요약, 핵심 인사이트, 권장 목차, 리스크 및 한계, 후속 분석 제안, 확인 필요 사항을 생성한다.

## 선택 업무 과업

- **업무명:** 데이터 분석 결과 기반 보고서 개요 생성
- **목표:** 분석 결과를 경영진 또는 실무 리더가 이해하기 쉬운 보고서 구조로 변환
- **주요 출력:** 보고서 제목, 한 줄 요약, 핵심 인사이트, 목차, 리스크 및 한계, 후속 분석 제안, 확인 필요 사항

## 최종 선정 모델

최종 선정 모델은 **ChatGPT**이다.

선정 이유는 다음과 같다.

1. 입력 데이터에 포함된 수치와 관찰된 패턴을 비교적 정확하게 반영하였다.
2. 요청한 출력 형식을 가장 안정적으로 준수하였다.
3. 가격, 경쟁사, 광고비처럼 제공되지 않은 정보를 단정하지 않고 확인 필요 항목으로 분리하였다.
